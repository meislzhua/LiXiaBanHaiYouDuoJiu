#开发文档
开发的注意事项将写在这里,以供参考

## 系统设置
系统设置在保存在浏览器的`localStorage`中(注意自行转换格式)

|设置项|类型|说明|示例|
|:-------------:|:-------------|:--------------|:------------:|
|time_xiaban|String|下班时间|"18:00"|

## 命令行
### Debug
开发时,可以使用本命令辅助开发
在页面内的html代码会实时刷新
``` bash
$ npm run dev
```

### Build
构建命令,可以生成构建后的代码
``` bash
$ npm run build
```
